# Week 8: 长上下文管理与安全工程

## 🎯 目标
攻克长上下文瓶颈，构建防注入、脱敏、审计管道。

## 🛠️ 每日实操
- D1: Token 爆炸 / 注意力稀释 / KV Cache 瓶颈分析
- D2: 上下文压缩 (滑动窗口 + 摘要链)
- D3: OWASP LLM Top 10 防御 (Prompt 注入/越权)
- D4: PII 脱敏 (正则/NLP) + 审计日志
- D5: 安全中间件 (请求过滤/沙箱执行)
- D6: 红队测试 (10 种 Jailbreak 用例)
- D7: 安全白皮书 + 数据保留/擦除策略

## ✅ 验收标准
- 长对话压缩意图保留率 >90%
- 注入拦截率 >95%
- 审计日志完整可追溯
