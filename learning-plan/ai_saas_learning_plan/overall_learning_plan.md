# 🎯 12周 AI-Native SaaS 工程师全景路线图

本计划以**单项目持续演进**为主线，每周叠加新能力，最终交付生产级 AI SaaS。

| 周次 | 核心主题 | 关键技术栈 | 核心交付物 |
|:---:|:---|:---|:---|
| W1 | AI-Native 环境 & SaaS 骨架 | Cursor, FastAPI, Vue3, LangChain | 异步 API + 基础 Agent + Docker |
| W2 | 多 Agent 编排与状态机 | LangGraph, CrewAI, State Schema | 可路由/可中断的多 Agent 系统 |
| W3 | RAG 管道与检索评估 | BGE-M3, Milvus/Chroma, RAGAS | 企业级知识库 RAG + 评估报告 |
| W4 | 私有化部署与推理加速 | vLLM, Ollama, AWQ/GPTQ, LiteLLM | 本地模型高吞吐 API 网关 |
| W5 | 领域微调与 LoRA 实战 | Unsloth, PEFT, QLoRA, Alpaca | 垂直领域 LoRA 模型 + 对比测试 |
| W6 | 小程序端接入与 UI/UX | Taro/Uniapp, Zustand, Figma | 微信端完整交互 + 跨端同步 |
| W7 | 高并发与流式优化 | WebSockets/SSE, Celery, k6 | 支持 500+ QPS 的流式服务 |
| W8 | 长上下文与安全工程 | Context Compression, OWASP LLM | 防注入/防越权的安全管道 |
| W9 | CI/CD 与生产监控 | GitHub Actions, Prometheus, Grafana | 自动化构建部署 + 可观测性 |
| W10| AI-Native 工作流标准化 | Prompt 版本库, 成本追踪 | 企业级 AI 开发规范 + Prompt 库 |
| W11| 作品集打磨与开源贡献 | GitHub 优化, Tech Blog, PR | 高 Star 潜力仓库 + 技术文章 |
| W12| 面试实战与系统设计 | 白板架构、故障排查、话术 | 模拟面试记录 + 架构手稿 |

> 📌 每周基于上周 `main` 分支迭代，最终仓库即为完整生产级项目。
